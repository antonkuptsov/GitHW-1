Git HW

git config --global user.name "name" - встановити name користувача

config --global user.email- встановити email користувача

git init- створити новий репозиторій

git status- показує стан робочого каталогу і розділу проіндексованих файлів

git add- додає вміст робочої директорії в індекс

git commit- збереження змін проіндексованих файлів

git log - показати історію комітів

git checkout- повернутись до останнього коміту

git branch- показати всі гілки

git push- відправити локальні зміни до origin у гілку

git pull- підтянути останні зміни з origin з переміщенням

git merge- мерджити 2 гілки в одну з збереженням історії комітів гілки з якою мерджимо

git clone- вибір уснуючого репозиторію для подальшого створення його клону чи копії

.gitignore файл- служить для вказівки в ньому файлів і папок, які необхідно приховати від системи контролю версій git. Як правило, приховують файли або папки, які автоматично генерують свій вміст, або мають конфігураційні параметри, які можуть відрізнятися у тих, хто співпрацює над проектом.
